## Running Locally

$ node --version
v10.15.3

$ npm install

Run database locally (see Database section)

To start locally, run the following script:
```
$ ./r
```
http://localhost:8100/


## Database

Postgres 9 or later

create database
run init.sql in scripts/postgresql/database-setup/ to create all tables
config local.yml in /config/
